<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

Docker or Visual Studio 2022

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/miczek-r/SM_Projekt_API.git
   ```
   #For Docker
2. Install NPM packages
   ```sh
   npm install
   ```
   #For Visual Studio 2022
4. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
